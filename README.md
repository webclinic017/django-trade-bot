### Getting started


Create virtual env:
`python3 -m venv env`
Activate:
`source ./env/bin/activate`

Requirements:

- Django (3.1.7)
- django-rest-swagger (2.2.0)
- djangorestframework (3.12.2)
- drf-yasg (1.20.0)
- python-binance (0.7.9)
